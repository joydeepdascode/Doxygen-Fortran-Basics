\begin{DoxyEnumerate}
\item Install Doxygen from website(for Windows), from Terminal(for Linux).
\item Install Doxywizard from website or terminal accordingly.
\item Open Doxywizard, and in the Wizard Column, under Topics row, fill the form.
\item Click Next.
\item In the Mode row, select your desired programming language.
\item Click Next, check for H\+T\+ML, Latex.
\item Click Next, check for diagrams to generate.
\item In the Expert Column, go to Project row, and enable \char`\"{}\+J\+A\+V\+A\+D\+O\+C A\+U\+T\+O\+B\+R\+I\+E\+E\+F\char`\"{}.
\item In the Build row, check \char`\"{}\+E\+X\+T\+R\+A\+C\+T\+\_\+\+A\+L\+L\char`\"{}. Uncheck \char`\"{}\+S\+H\+O\+W\+\_\+\+U\+S\+E\+D\+\_\+\+F\+I\+L\+E\+S\char`\"{}.
\item In the H\+T\+ML row, check \char`\"{}\+G\+E\+N\+E\+R\+A\+T\+E\+\_\+\+T\+R\+E\+E\+V\+I\+E\+W\char`\"{}. And, Uncheck \char`\"{}\+D\+I\+S\+A\+B\+L\+E\+\_\+\+I\+N\+D\+E\+X\char`\"{}.
\item In the Dot row, check \char`\"{}\+U\+M\+L\+\_\+\+L\+O\+O\+K\char`\"{}.
\item Finally, Click on Run column.
\item Go to output folder, and open index.\+html to view the output.
